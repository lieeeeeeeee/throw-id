# ダーツ自己紹介カードメーカー

ダーツの自己紹介カードをブラウザで作成し、**615×870（PNG）**で保存できるWebアプリです。  
Xに **`#ダーツ好きと繋がりたい` `#ダーツ自己紹介カード`** と一緒に投稿して交流に使えます。

## 特徴
- **入力 → プレビュー → PNG出力**までクライアント完結
- **年齢・性別は必須**（未入力だとダウンロード不可）
- **ライブテーマ/ファンダーツ/ダーツカード画像は任意**（未入力でもカード側は自動でプレースホルダー表示）
- レーティング等は**短い自由記述**で入力可能

## 開発

```bash
npm install
npm run dev
```

`http://localhost:3000` を開いてください。

## 使い方
1. 左側のフォームに入力
2. 右側のプレビューで確認
3. 「PNGをダウンロード」→ 画像を保存
4. Xに画像を添付して投稿（例）

```
#ダーツ好きと繋がりたい #ダーツ自己紹介カード
```

## デプロイ（Vercel）
- 通常はそのままデプロイ可能です。
- メタ情報のURL確定をしたい場合は、Vercelの環境変数に `NEXT_PUBLIC_SITE_URL`（例：`https://your-app.vercel.app`）を設定してください。

## 注意
- 画像はブラウザ内で扱います（サーバにアップロードしない想定）。
